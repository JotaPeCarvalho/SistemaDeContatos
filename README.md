## Contact System
A contact system capable of adding contacts and users, having administrator and comum user profiles.



## Technology 

Here are the technologies used in this project.

* HTML
* CSS
* Bootstrap
* JavaScript
* JQuery
* C# .NET
* ASP.NET MVC
* EntityFramework
* SQLServer

## Services Used

* Github
